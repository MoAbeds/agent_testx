# Mojo SaaS

This is a Next.js project scaffolded for Mojo SaaS.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Structure

*   `app/dashboard`: The main dashboard view.
*   `components/TerminalFeed.tsx`: Live scrolling log component.
*   `components/StatsCard.tsx`: Reusable metric cards.
*   `components/DiffViewer.tsx`: Comparison view for request inspection.
